# Pembahasan Materi

## 1. Software Reverse Engineering

Software Reverse Engineering merupakan proses menganalisis perangkat lunak tanpa memiliki source code asli. Analisis dilakukan terhadap file executable atau binary untuk mengetahui bagaimana program bekerja.

Beberapa tujuan Software Reverse Engineering antara lain:

- Memahami logika program.
- Mengetahui struktur executable.
- Melakukan analisis malware.
- Menemukan kerentanan keamanan.
- Mendokumentasikan aplikasi lama.

Saya memahami bahwa jenis Reverse Engineering ini merupakan yang paling banyak digunakan dalam bidang keamanan siber.

---

## 2. Hardware Reverse Engineering

Hardware Reverse Engineering dilakukan dengan cara mempelajari perangkat keras untuk mengetahui desain, fungsi, maupun cara kerja setiap komponennya.

Beberapa penerapannya meliputi:

- Analisis papan sirkuit (PCB).
- Identifikasi komponen elektronik.
- Rekonstruksi desain perangkat.
- Pengembangan perangkat yang kompatibel.

Materi ini menunjukkan bahwa Reverse Engineering tidak hanya terbatas pada software, tetapi juga banyak digunakan pada perangkat elektronik.

---

## 3. Protocol Reverse Engineering

Protocol Reverse Engineering digunakan untuk memahami bagaimana suatu perangkat melakukan komunikasi dengan perangkat lain.

Proses analisis dilakukan terhadap:

- Struktur paket data.
- Header komunikasi.
- Format pesan.
- Mekanisme pertukaran data.

Teknik ini sangat berguna ketika dokumentasi protokol tidak tersedia sehingga komunikasi antar perangkat tetap dapat dipahami.

---

## 4. Perbandingan Ketiga Jenis Reverse Engineering

Setiap jenis Reverse Engineering memiliki fokus analisis yang berbeda.

| Jenis | Objek Analisis | Tujuan |
|--------|----------------|--------|
| Software Reverse Engineering | Executable/Binary | Memahami logika program |
| Hardware Reverse Engineering | Perangkat elektronik | Memahami desain dan fungsi perangkat |
| Protocol Reverse Engineering | Paket komunikasi | Memahami mekanisme komunikasi data |

Dari materi ini saya mengetahui bahwa ketiga jenis tersebut saling melengkapi sesuai kebutuhan analisis.
